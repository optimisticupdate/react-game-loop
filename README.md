# React Game Loop

Example for building a game loop with react.

## Getting Started

```sh
# Development
$ npm run dev

# App listening on http://127.0.0.1:1234

# Build
$ npm run build
```

## Recommended Editor and Plugins

**Editor used:** VSCode  
**Plugins:**

- Eslint
- PostCSS Language Support
- Prettier
- Tailwind CSS IntelliSense
